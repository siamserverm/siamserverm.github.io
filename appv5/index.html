ผู้ใช้ / โปรไฟล์งาน Workspace

            
ใช้แยกงานไม่ให้ Landing Page และสถิติปนกัน หลายคนใช้ Workspace ID เดียวกันเพื่อดูสถิติชุดเดียวกันได้

            
            
            
            
            

            

              
รายการโปรไฟล์งาน

              
เลือกจากรายการนี้ได้เลย ไม่ต้องจำ Workspace ID เอง

              

            

            
ตัวอย่าง: งาน A ใช้ workspace-a, งาน B ใช้ workspace-b สถิติจะถูกเก็บคนละโฟลเดอร์ใน Firebase

          

          

            
ตั้งค่า Firebase Realtime

            
ใช้ Firebase Realtime Database เป็นฐานข้อมูลนับคลิกจริง ไม่ต้องใช้ Apps Script / Cloudflare / GA4 Realtime API

            
            
            
ยังต้องการใช้ Google Analytics ไหม?

            

            
Database ที่ตั้งไว้ให้แล้ว: affiliate-landing-stats

            
ตอนส่งออก ZIP / เผยแพร่ ระบบจะฝัง Firebase URL นี้ลงใน Landing Page เพื่อให้นับคลิกจากทุกเครื่องรวมกัน

          

          
วิธีใช้งานเร็ว ๆ
  1. เพิ่มสินค้าและใส่ slug เช่น a
  2. ใส่รูป + ลิงก์ Shopee/Lazada
  3. ถ้าช่อง Google Analytics ID ของสินค้าเว้นว่าง ระบบจะใช้ค่าเริ่มต้นจากหน้านี้
  4. กดส่งออก ZIP หรือเผยแพร่ผ่าน GitHub Pages
  5. เว็บจะได้ลิงก์แบบ /a และเริ่มนับคลิก Shopee/Lazada เข้า Firebase Realtime
  6. รูปจะถูกเก็บใน IndexedDB ไม่กินพื้นที่ localStorage แล้ว
